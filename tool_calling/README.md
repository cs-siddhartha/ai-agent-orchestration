## Tool Calling in AI agents

Tool Calling in ai agents.

example.py contains the raw orchestration logic
example_langchain.py contains the langchain framework logic. 

Obvious inference from the examples is that langchain abstracts the logic of how the agent loops works. 

The example has plenty of print statements added to make sure that devs can understand the flow of how the agentic loop is working and how the llm responds requesting a tool for action.
