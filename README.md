# ai-agent-orchestration
Agent Orchestration patterns from scratch.

Implementation of various ai agent orchestration recipes from scratch without the use of any framework. Each folder/example contains the orchestration breakdown along with a framework example (langchain/crewai) to show usage of how the agentic loop is structured.
